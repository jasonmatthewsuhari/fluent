# Support

Fluent is pre-alpha software. It is not ready for dependable daily assistive use and should not be relied on for emergency, medical, or safety-critical workflows.

## Getting Help

- Use GitHub issues for bugs, feature requests, and accessibility feedback.
- Use discussions, if enabled, for broader product questions and ideas.
- For security concerns, follow [SECURITY.md](SECURITY.md).

## Accessibility Feedback

Feedback from disabled users, elders, rehabilitation professionals, caregivers, and assistive-technology users is especially valuable.

Helpful reports include:

- What you were trying to do.
- What made the experience difficult.
- Which input methods or assistive tools you use.
- Screenshots or recordings, if comfortable and safe to share.
- Any privacy concerns.

## What Fluent Is Not Yet

Fluent is not currently a certified assistive device, medical product, or replacement for human support. The project is still validating the first text-driven agent loop.
