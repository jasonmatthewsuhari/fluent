# Vision

Fluent is an open-source desktop AI agent for accessible computer control.

The project exists for people who need alternatives to traditional mouse and keyboard interaction, including disabled users, elders, and people in rehabilitation. Fluent should make computer control more direct, legible, and forgiving by letting users express intent and supervise the result.

## Product Direction

The first version uses text input only. This validates the agent loop before adding more input modes.

Future versions should support:

- Voice input.
- Switch access.
- Camera or gesture input.
- Eye tracking.
- Other assistive input hardware and software.
- Screenshot or vision-language-model reasoning where automation context is insufficient.

## Principles

- The user remains in control.
- Automation is visible before and after meaningful actions.
- Risky actions require confirmation.
- Stop and pause controls are always reachable.
- The app is useful with user-supplied AI provider keys.
- Accessibility decisions are product decisions.

## Non-Goals for the First Pass

- Full multimodal input.
- Clinical or medical claims.
- Emergency or safety-critical reliability.
- Paid-only core workflows.
- A general-purpose marketing landing page inside the app.
